# Sign
To sign a document, us `gpg2 --armor --default-key 2A4E0E15 --clearsign ./index.html`

To verify, remove the lines involving the signature, save them in a file (index.html.sig) and then run `gpg2 --verify index.html.sig index.html`

To verify that the signature was signed with my private key, download the attached public key and verify the signature with `gpg2 --no-default-keyring --keyring ../../../public.key --verify ./index.html.sig`
